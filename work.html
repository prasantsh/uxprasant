Case studies - UI/UX work, start to finish

Three interface projects taken from problem to shipped design - the constraint, the thinking, and what changed. Broader design work follows below.

№ 01

UI Designer 2024 - Present Mobile App

Hypnotik FX - smart lighting control

Challenge

Powerful LED controllers - Audio Reactive, X-Pro - shipped with an app so dense that new users could not get from unboxing to a working light scene. Segments, effects, palettes and sync all competed for attention at once.

Approach

Rebuilt the control flow around a task-first hierarchy - effects and palettes up front, wiring and segment setup behind progressive disclosure. Paired the interface with a documentation set that mirrors each in-app step.

Outcome

A calmer, learnable app where audio-reactive setups and presets are a few taps away, backed by a reusable component and documentation system the team extends per feature.

№ 02

Creative / UI Designer 2023 - 2024 Web Platform

RepairMate AU - device repair, booked

Challenge

A device-repair service in Australia needed a booking and status experience that felt trustworthy and fast for customers, and legible for the staff managing jobs behind it.

Approach

Structured the booking journey step by step, designed clear repair-status states, and unified the brand's visual system across the customer-facing site and its campaign assets.

Outcome

A cohesive customer experience with a consistent brand voice carried from product screens through to marketing - one system, not scattered pieces.

  • Web
  • Booking Flow
  • Brand System
  • Campaign

№ 03

UI / Visual Designer 2022 - 2023 Web

Global - product site & system

Challenge

An emerging brand needed a web presence that read as credible and current, with a layout system flexible enough to grow as the offering expanded.

Approach

Set a clear typographic and grid foundation, designed a responsive landing experience around it, and kept every section on a shared set of components and spacing rules.

Outcome

A responsive site that holds together across breakpoints and stays consistent as new sections are added - design decisions made once, reused everywhere.
